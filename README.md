 🎯 Number Guessing Game in C
A simple command-line number guessing game built using C.

🚀 Features
- Random number generation (1–100)
- User input handling
- Hints (Higher / Lower)
- Tracks number of attempts

🛠️ Technologies Used
- C Programming
- Standard Libraries: stdio.h, stdlib.h, time.h

▶️ How to Run

1. Compile the code:
   ```bash
   gcc main.c -o game
2.Run the program:
./game
