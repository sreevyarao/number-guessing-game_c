
 # 🎯 Smart Number Guessing Game in C
<br>
An interactive command-line number guessing game built using C with multiple difficulty levels, smart hints, and a scoring system.
<br>
This project was created to strengthen my understanding of core programming concepts like loops, conditionals, and random number generation.



🚀 Features

- 🎮 Multiple Difficulty Levels:
  - Easy (1–50, 10 attempts)
  - Medium (1–100, 7 attempts)
  - Hard (1–200, 5 attempts)

- 🧠 Smart Hints:
  - "Very close" (within ±3)
  - "Close" (within ±10)
  - Higher / Lower guidance

- 🏆 Scoring System:
  - Score based on attempts taken
  - High score tracking during runtime

- 🔁 Replay Option:
  - Play multiple rounds without restarting

🛠️ Technologies Used

- C Programming
- Standard Libraries:
  - `stdio.h`
  - `stdlib.h`
  - `time.h`


▶️ How to Run

1. Compile the program:
```bash
gcc main.c -o game
2.Run the program:
